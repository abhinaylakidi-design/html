CSS Combinator (Combination) Selectors

Combinator selectors are used to select elements based on their relationship with other elements. They help apply styles more precisely.

1. Descendant Selector (space)

The descendant selector selects all elements that are inside another element, no matter how deeply nested.

This paragraph is inside the div

This paragraph is deeply nested

2. Child Selector (>)

The child selector selects only direct children of a specified element.

This is a direct child (styled)

This is not a direct child (not styled)

3. Adjacent Sibling Selector (+)

The adjacent sibling selector selects the first element that comes immediately after another element.

Heading

This paragraph is immediately after h3 (styled)

This paragraph is not adjacent (not styled)

4. General Sibling Selector (~)

The general sibling selector selects all sibling elements that come after a specified element.

Heading

This paragraph is styled

This paragraph is also styled

Summary

Combinator selectors allow developers to target elements based on hierarchy and position, making CSS more powerful and flexible.
