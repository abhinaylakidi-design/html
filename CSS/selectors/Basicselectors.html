CSS Selectors

CSS selectors are used to select HTML elements and apply styles to them. They tell the browser which HTML elements should be styled.

1. Element Selector

The element selector selects HTML elements based on their tag name.

This is a span element styled using an element selector.

2. Class Selector

The class selector selects elements with a specific class name. It is reusable and starts with a dot (.).

This text uses a class selector

3. ID Selector

The ID selector selects a unique element. It starts with a hash (#) and should be used only once.

This text uses an ID selector

4. Group Selector

The group selector styles multiple elements at the same time using commas.

This is Heading 3

This is Heading 4

5. Universal Selector

The universal selector (*) selects all elements.

This paragraph is italic

This span is also italic

6. Attribute Selector

The attribute selector selects elements based on an attribute and its value.

Conclusion

CSS selectors are a powerful way to target HTML elements. By using the right selector, developers can write clean, reusable, and efficient styles.
