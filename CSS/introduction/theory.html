Introduction to CSS

Importance of CSS

CSS (Cascading Style Sheets) is used to control the appearance of web pages. Without CSS, websites would look plain and unstructured. CSS makes websites attractive, readable, and user-friendly.

  • Improves the visual design of web pages
  • Makes content easy to read and understand
  • Helps create responsive layouts
  • Separates content from design

Why CSS is Used

CSS is used to style HTML elements such as text, images, buttons, and layouts. It allows developers to apply the same styles across multiple pages, saving time and effort.

  • To add colors, fonts, and spacing
  • To design layouts and page structure
  • To maintain consistency across pages
  • To reduce repeated code

1. Inline CSS

Inline CSS is written directly inside an HTML element using the style attribute. It applies only to that element.

2. Internal CSS

Internal CSS is written inside the <style> tag in the head section of the HTML file. It affects only one page.

3. External CSS

External CSS is written in a separate .css file and linked to the HTML document. It is reusable for multiple pages.
