CSS Box Model

Every HTML element is a rectangular box made up of these layers

Margin (space outside the element)
Border (outline of the element)
Padding (space inside the border)
Content (Text / Image / Element)

box-sizing: border-box

With box-sizing: border-box, the width includes padding + border

This box has:
Width: 300px
Padding: 20px
Border: 5px
