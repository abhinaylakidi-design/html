CSS Position Property – Explained with Structure & Styling

1. position: static (Default)

Static is the default position for all HTML elements. The element follows the normal document flow. top, left, right, bottom will NOT work.

Static
Use static when you don’t need any special positioning.

2. position: relative

Relative positioning moves the element relative to its original position. The space it originally occupied is still preserved.

Relative
Mostly used as a reference for absolute elements.

3. position: absolute

Absolute positioning removes the element from the normal document flow. It positions itself relative to the nearest positioned ancestor (relative, absolute, fixed).

Absolute
Commonly used for popups, badges, and overlays.

4. position: sticky

Sticky behaves like relative until a certain scroll position is reached, then it behaves like fixed.

Sticky Header

Scroll the page to see the sticky effect in action.

Used for headers, menus, and section titles.

5. position: fixed

Fixed positioning places the element relative to the viewport. It stays in the same position even when scrolling.

The red box at the bottom-right of the screen is fixed.
